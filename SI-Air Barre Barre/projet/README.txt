Pour ce projet, vous devez installer :
 - Nodejs: https://nodejs.org/en/
 - npm

Ensuite, dans un terminal une fois déplacé à l'emplacement du dossier, 
installer les modules avec 'npm install [nom du module]' :
	- express
	- serve-static
	- http
	- fs
	- nodemon

Enfin, démarrer le serveur avec : nodemon serveur.js
